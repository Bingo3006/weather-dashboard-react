# Weather Dashboard React

A modern weather dashboard built with React and OpenWeather API.

## Features
- Current weather
- 5-day forecast
- Dynamic weather themes
- Responsive UI
- Weather icons
- Error handling
- Loading animations

## Technologies Used
- React
- Vite
- JavaScript
- OpenWeather API
